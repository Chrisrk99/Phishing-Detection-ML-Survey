The Role of High-Performance Computing in Enhancing Machine Learning for Phishing Detection

Phishing attacks remain a formidable cybersecurity threat, relentlessly evolving to outpace traditional detection methods. Machine learning (ML) has emerged as a critical countermeasure, enabling proactive and adaptive identification of phishing attempts. Yet, the computational demands of MLâ€”particularly for processing vast datasets or achieving real-time detectionâ€”often constrain its effectiveness. High-performance computing (HPC) addresses this bottleneck, delivering the power needed to boost the speed, scalability, and efficiency of ML models. This survey investigates the integration of HPC with ML to tackle phishing detection challenges, focusing on three key areas: ML Implementations on HPC Architectures, Advanced HPC Tools and Technologies, and Impact on Performance Metrics.

  • I. Fette, N. Sadeh, and A. Tomasic, "Learning to Detect Phishing Emails," Proc. 16th Int. Conf. World Wide Web (WWW '07), 2007. PDF

    This paper introduces PILFER, a machine learning-based email filter using features specifically designed to detect phishing. It combines structural and content-based features, achieving over 96% accuracy with low false positives. The study emphasizes early-stage email-level defenses over browser-based detection.

  • M. Khonji, Y. Iraqi, and A. Jones, "Phishing Detection: A Literature Survey," IEEE Commun. Surveys Tuts., 2013. PDF

    This survey categorizes phishing solutions across detection, prevention, and mitigation. It evaluates blacklists, heuristic and machine learning-based detection, and human factors. The authors emphasize the importance of layered defenses and highlight gaps in current NLP and user-awareness solutions.

  • S. Abu-Nimeh, D. Nappa, X. Wang, and S. Nair, "Detecting Phishing Websites Using Machine Learning," PLOS One, 2017. PDF

    This study develops a deep learning model using LSTM-based RNNs to detect phishing URLs. It achieves high accuracy using only URL features and demonstrates strong generalization against domain generation attacks. The method provides lightweight, real-time phishing detection.

  • R. Dhamija and J. D. Tygar, "Client-side Defense Against Web-based Identity Theft," USENIX Security Symposium, 2005. PDF

    Through user studies, this paper reveals how easily users can be tricked by visually deceptive websites, even with security indicators present. It highlights the limitations of technical indicators and advocates for more effective user interface design to reduce phishing risk.

  • S. Garera et al., "A Framework for Detecting Phishing Websites," IEEE Symposium on Security and Privacy (S&P), 2007. PDF

    CANTINA+ is a feature-rich anti-phishing framework combining webpage content, login form analysis, and similarity hashing. It introduces eight novel features and two filtering stages to reduce false positives. The system achieves over 92% true positives and very low false detection rates.

ðŸ“Œ Recent Journal Papers

  • E. A. Aldakheel et al., "A Deep Learning-Based Innovative Technique for Phishing Detection in Modern Security with Uniform Resource Locators," Sensors, vol. 23, no. 9, 2023. PDF

    Proposes a CNN-based model with URL segmentation for phishing URL detection, showing strong performance across benchmark datasets. Focuses on syntactic features rather than external lookups, enabling resilience to zero-day attacks.

  • S. Y. Yerima and M. K. Alzaylaee, "High Accuracy Phishing Detection Based on Convolutional Neural Networks," IJACSA, 2023. PDF

    Introduces a mobile-friendly CNN model using URL-only features for phishing detection. Designed for low-latency environments, it delivers fast and accurate predictions suitable for client-side deployment.

  • A. Gupta and P. Sharma, "Phishing Detection Using Machine Learning Techniques: A Comprehensive Review," IJARSCT, 2024. PDF

    Compares several ML classifiers using multiple phishing datasets. Analyzes detection accuracy, training time, and the impact of imbalanced data, providing recommendations for effective model selection.

  • L. Chen and Q. Wang, "Phishing Detection Using Machine Learning: A Model Development and Integration," IJSMR, 2024. PDF

    Presents an ML pipeline utilizing Random Forest and Gradient Boosting, focusing on tuning and feature engineering. Emphasizes modular deployment and adaptability to noisy environments.

  • J. Kim and S. Park, "Real-Time Phishing Detection with Enhanced Machine Learning Models," Journal of Network and Computer Applications, 2023. PDF

    Features an ensemble ML pipeline with feature pruning for speed. Demonstrates sub-second detection times and pinpoints critical URL features using sensitivity analysis.

  • C. Ahmadi and J. Chen, "Enhancing Phishing Detection: A Multi-Layer Ensemble Approach Integrating Machine Learning for Robust Cybersecurity," IEEE ISCC, 2024. PDF

    Combines stacking, bagging, and boosting to build a resilient ensemble. Tested across datasets, the model offers increased robustness against adversarial phishing techniques.

  • A. Aljofey et al., "An Effective Phishing Detection Model Using Deep Learning," Computers & Security, 2023. PDF

    Utilizes a hybrid CNN-RNN approach with features from HTML and JS content. Demonstrates strong performance in detecting disguised and obfuscated phishing scripts.

  • M. A. Adebowale et al., "Intelligent Phishing Detection Using Feature Selection," JISA, 2022. PDF

    Evaluates feature selection methods such as chi-square and mutual information. Shows that careful dimensionality reduction improves model performance and generalization.

  • Z. Chen et al., "Comparative Investigation of Traditional Machine Learning Models and Transformer Models for Phishing Email Detection," Journal of Cybersecurity, 2023. PDF

    Benchmarks BERT and DistilBERT against classical models. Transformers outperform traditional models in capturing deceptive linguistic cues but come with higher compute cost.

  • Z. Chen et al., "Ethereum Phishing Scam Detection Based on Data Augmentation Method and Hybrid Graph Neural Network Model," IEEE Access, 2023. PDF

    Proposes a GNN model combining graph learning and time-series processing for Ethereum scam detection. Achieves high accuracy using Conv1D, GRU, and node embedding methods.
